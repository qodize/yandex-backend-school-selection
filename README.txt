Сервис реализован на Django с использованием СУБД PostgreSQL
Для деплоя проекта на сервер требуется:
Клонировать репозиторий $ sudo git clone
Создать виртуальное окружение в корне репозитория $ sudo python3 -m venv venv
Активировать виртуальное окружение $ sudo source venv/bin/activate
Установить все внешние библиотеки $ pip install -r requirements.txt 
По умолчанию используется моя удаленная база данных на хостинге heroku, если вы желаете использовать свою базу данных,
введите все данные о ней в переменную DATABASES в файле yandex_backend/settings.py
Вы можете запустить проект без использования веб-сервера (небезопасно в проде) $ python3 manage.py runserver 0.0.0.0:8080
Рекомендуется использовать веб-сервер (apache2 или nginx или другие) совместно с WSGI, встроенным в Django (инструкции по развертыванию сервера можно найти в документации к Django)
В частности, я использую apache2
